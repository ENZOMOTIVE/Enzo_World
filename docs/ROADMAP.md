# Roadmap

## Phase 1: Local Arena Foundation

- Game catalog
- Room list and filters
- Create/join/leave rooms
- Player readiness
- Start match
- Chess legal moves with `chess.js`
- Carrom scoring prototype
- Room chat and events

## Phase 2: Real Multiplayer

- Auth and player profiles
- Room service API
- WebSocket room subscriptions
- Server-authoritative game state
- Reconnect and timeout handling
- Match result persistence

## Phase 3: Commercial Layer

- Wallet ledger
- Entry fees and rewards
- Store compliance review
- Abuse prevention
- Reporting and moderation tools
- Push notifications
- Analytics and crash reporting

## Phase 4: Production Polish

- Real icon and splash assets
- Onboarding
- Ranked/unranked queues
- Friends and private invites
- Match history
- Game tutorials
- Accessibility pass
